Family Tree
Создайте своё генеалогическое древо
Family Tree — это приложение для Android для управления генеалогическими деревьями.
Оно распространяется по модели freemium: почти все функции бесплатны, кроме одной (объединение деревьев), доступной по премиум-подписке.

Возможности
С Family Tree вы можете:

Создавать древо с нуля, добавляя имена, даты, места, события, фото и источники.

Импортировать существующее древо через файл GEDCOM и редактировать его.

Экспортировать созданное древо (также в GEDCOM) для загрузки в другие программы.

Делиться деревом с родственниками, принимая или отклоняя их изменения.

Экспортировать схему в PDF или PNG.

Минимальная поддерживаемая версия — Android 5 Lollipop (API 21), максимальная — Android 15 Vanilla Ice Scream (API 35).
Структура данных максимально соответствует стандарту GEDCOM 5.5.1 и, по возможности, 5.5.5.
В основе приложения лежит библиотека Gedcom 5 Java от FamilySearch.

Ограничения
Код из этого репозитория собирается в рабочую версию, но с ограничениями:

Подпись приложения: при установке поверх подписанной версии сохранённые деревья будут потеряны.

Серверный аккаунт: функция совместного редактирования недоступна.
